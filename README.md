# SC-Relocale
A complete overhaul of all displayed texts in Simplified Chinese for No Man's Sky, now work in progress.

## 构建模组
- 下载 PSArcTool；
- 无需打开上述应用程序。直接将 LANGUAGE 文件夹拖拽至 PSArcTool.exe；
- 得到的 pak 文件即为成品模组，可按一般流程加载。
